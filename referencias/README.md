# Referencias 📋

## Descripción

Colección de fuentes primarias, secundarias y recursos digitales para investigación profunda.

## Tipos de Referencias

### 📄 Fuentes Primarias
Documentos originales, actas, cartas, proclamas, documentos gubernamentales.

### 📚 Fuentes Secundarias
Análisis, interpretaciones, historiografía, comentarios académicos.

### 💻 Recursos Digitales
Bases de datos, archivos digitales, colecciones en línea, repositorios académicos.

## Acceso

- Enlaces a documentos digitalizados
- Ubicación de archivos físicos
- Instrucciones de acceso
- Consideraciones de copyright

## Cómo Contribuir

Si conoces una fuente valiosa:
1. Verifica su autenticidad
2. Obtén enlace o información de ubicación
3. Agrega a la carpeta correspondiente
4. Incluye metadatos completos

Las fuentes son la base del conocimiento riguroso.
