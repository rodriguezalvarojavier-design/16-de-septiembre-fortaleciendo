# Materiales Educativos 🎓

## Descripción

Esta carpeta contiene recursos pedagógicos diseñados para diferentes niveles educativos y contextos de aprendizaje.

## Subcarpetas Sugeridas

### 📖 Guías Pedagógicas
Estrategias y metodologías para la enseñanza del tema.

### 📝 Materiales Docentes
Recursos listos para usar en el aula.

### ⏱️ Cronologías
Líneas de tiempo y secuencias de eventos clave.

### 👥 Biografías
Vidas y legados de personajes históricos relevantes.

## Niveles Educativos

- **Primaria (6-12 años)** - Conceptos básicos y narrativas accesibles
- **Secundaria (12-18 años)** - Análisis más profundo y pensamiento crítico
- **Universitaria (18+)** - Investigación académica y perspectivas complejas
- **Público general** - Recursos de divulgación científica

## Cómo Usar

1. Selecciona el nivel educativo
2. Elige el tipo de recurso
3. Adapta según tus necesidades específicas
4. Comparte feedback para mejora continua
